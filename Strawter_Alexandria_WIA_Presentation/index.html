What is Tooltipster?

Tooltipster is a simple and easy jQuery plugin that allows you to add fun stylish tooltips to your page. It was developed by Caleb Jacob and Louis Ameline.

Tooltipster is supported by...

  1. Chrome
  2. Firefox
  3. Opera
  4. Safari
  5. Internet Explorer 6+

Tooltip Themes

With tooltipster you can use the predefined styles or even create your own to match the style of your page.


  • Default
  • Light
  • Borderless
  • Punk
  • Noir
  • Shadow
  • Custom

How to use it

To start off using tooltipster after you have the downloaded files you need to make sure to include the css and the js in your html <head> element

The js file should be entered after jQuery file for it to properly work

Placing a tooltip in your html

To actually create a tooltip you want to create a class of "tooltip" and a title which would contain the content that is shown


Example

Hello I am a paragraph with a tooltip

<p class="tooltip" title="I am the tooltip!">Hello I am a paragraph with a tooltip</p>

But wait...that doesn't look like any of the tooltips we've seen before!? That's because we haven't activated it yet

Activating the tooltip

You can create a new js file or you can enter your script right before the closing <body> tag. Just make sure to create a document ready function and then call tooltipster

You can use any element to call it but for this example we will call it by the class .tooltip
When you call tooltipster you can set the theme to whichever style you like best. In this case we are using the default 
$(document).ready(function() {
	$('.tooltip').tooltipster({
		theme: 'tooltipster-default'
	});
});

Making a tooltip into a link


Example

I wonder what this does...

I wonder what this does...
$('#linkable').tooltipster({
	contentAsHTML: true,
	interactive: true,
	animation: 'fall', //'fade', 'grow', 'swing', 'slide', 'fall'
	side: 'bottom' //'top', 'bottom', 'right', 'left'
});
Setting contentAsHTML changings the text from a string into actual html when set to true. interactive does just what it says if you want the user to interact with the tooltip in anyway set this to true. We also added animation in this one I chose the fall effect and placed it on the bottom. This makes the tooltip seem to be falling from the top of the page to the bottom of the element.

Clicking for a tooltip

Simple as it sounds you can use a click or keypress to open/close the tooltip


Example

Just click me!

Just click me!
$('#click').tooltipster({
	trigger: 'click' //'hover','click', 'custom'
});

$(window).keypress(function() {
	$('#click').tooltipster('hide');
});

Multiple tooltips

Example

Oh my look at all these tips! Which one should I read first?

Oh my look at all these tips! Which one should I read first?
var $four= $('#fourTips');
$four.tooltipster({
	content: 'Top', //set the content that would appear with each tooltip
	side: 'top', //set the side you want to show the tooltip
	theme: 'tooltipster-custom' //select the theme for the tooltip
});  //you could also add an animation for each tooltip
$four.tooltipster({
	content: 'Right',
	multiple: true,
	side: 'right',
	theme: 'tooltipster-shadow',
});
$four.tooltipster({
	content: 'Bottom',
	multiple: true,
	side: 'bottom',
	theme: 'tooltipster-custom'
});
$four.tooltipster({
	content: 'Left',
	multiple: true,
	side: 'left',
	theme: 'tooltipster-shadow'
});
